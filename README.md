## Contribute

### Getting Started

```bash
go get github.com/ahmedaly113/eltra-api
cd GOPATH/ahmedaly113/eltra-api
make install
```

Then copy/paste the `.env.sample` to `.env` and update the values.

### Run locally

```bash
make start
```

### Test

```bash
make test
```
